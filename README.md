# Kassenbuch

App für Kassenführung à la "Milchbüechli", geeignet für private Projekte oder kleine Vereine

https://kutskey-kassenbuch.share.connect.posit.cloud

Features:
- Anlassverwaltung
- Kontenverwaltung
- Buchungsverwaltung
- Import/Export der Daten als Excel
- Darstellung der Daten als Reactive Table im Dashboard
- Automatisierte Anlassabrechnung als PDF
- Automatisierter Kontoauszug als PDF
